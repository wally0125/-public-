資料檔:
water.csv  ---------------------- 每天水質資料
water_clean.csv  ---------------------- 資料清洗後的水質資料
tank.csv   ---------------------- 每天蝦隻所在水缸位置
grown.csv  ---------------------- 每隻蝦的卵巢成熟階段
weight.csv ---------------------- 每隻蝦的重量


執行"data_main.py"將上述資料檔合併且前處理後產生"dataset.csv"用來訓練的檔案

將"dataset.csv"放入"main.py"做訓練並產生結果

dataset.csv  ---------------------- data_main.py產生的檔案，只經過合併和清洗
data_final.csv ------------------ 將dataset.csv刪去異常值後的資料
data_predict_mix_7_7.csv -------- 使用前7天的資料預測後7天的資料集

data_main.py, data.py ---------------------- 用來合併和對資料作前處理
method.py ------------------------- 放訓練方法的地方
main.py ---------------------- 訓練和預測的地方
draw_plot.py ----------------- 用來視覺化結果
